Lesson 1: What is JavaScript?

JavaScript is a programming language used to create interactive and dynamic websites.

It runs in browsers and is widely used for web development, game development, and more.

Lesson 2: Variables and Data Types

JavaScript variables store data.


var name = "Larmel";  // Old way (Avoid using var)
let age = 19;         // Modern way (Recommended)
const country = "Philippines"; // Constant variable (Cannot be changed)

Lesson 3: Operators

JavaScript provides arithmetic and comparison operators.


console.log(10 + 5);  // Output: 15
console.log(10 > 5);  // Output: true

Lesson 4: Conditional Statements

Conditional statements allow JavaScript to make decisions based on conditions.


let age = 19;
if (age >= 18) {
console.log("You are an adult.");
} else {
console.log("You are a minor.");
}

Lesson 5: Loops

Loops repeat actions multiple times.

Example of For Loop:


for (let i = 1; i <= 5; i++) {
console.log("Iteration #" + i);
}

Lesson 6: Functions

Functions allow us to reuse code multiple times.


function greet(name) {
return "Hello, " + name + "!";
}
console.log(greet("Larmel"));

Lesson 7: Arrays

Arrays store multiple values in a single variable.


let colors = ["Red", "Green", "Blue"];
console.log(colors[0]); // Output: Red
console.log(colors.length); // Output: 3

Lesson 8: Objects (Key-Value Pairs)

Objects store multiple values inside a single variable using key-value pairs.

For example, if you want to store a person's name, age, and country:


let person = {
  name: "Larmel",
  age: 19,
  country: "Philippines"
};

console.log(person.name); // Output: Larmel

Explanation:

  • name - key for the person's name
  • age - key for the person's age
  • country - key for the person's country

The data is grouped into a single variable, making it more organized!

Lesson 9: DOM Manipulation

JavaScript can modify the content of a webpage using the **DOM (Document Object Model)**.

In the example below, clicking the button will change the text in the **heading**.

Original Title

Code Explanation:


function changeText() {
  document.getElementById("title").innerHTML = "Text Changed!";
}

How does it work?

  • document.getElementById("title") - Selects the element using its ID
  • innerHTML = "Text Changed!" - Replaces the content of the heading
  • When you click the button, the heading text will change.

Lesson 10: Event Listeners

With event listeners, you can create code that reacts to **user actions**, such as clicking a button.

Code Explanation:


document.getElementById("btn").addEventListener("click", function() {
  alert("Button Clicked!");
});

How does it work?

  • document.getElementById("btn") - Selects the button using its ID
  • addEventListener("click", function() {...}) - Adds an event listener
  • When you click the button, an alert box will appear!
