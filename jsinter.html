Lesson 1: Higher-Order Functions

Higher-order functions take another function as an argument or return a function.


let numbers = [1, 2, 3, 4, 5];

let squaredNumbers = numbers.map(function(num) {
return num * num;
});

console.log(squaredNumbers); // Output: [1, 4, 9, 16, 25]
  • map() applies a function to each element in the array.
  • The function squares each number and returns a new array.

Lesson 2: Closures

A closure is a function that remembers variables from its outer scope even after execution.


function createCounter() {
let count = 0;

return function() {
    count++;
    console.log(count);
};
}

let counter = createCounter();
counter(); // Output: 1
counter(); // Output: 2
  • count is preserved in memory even after createCounter() executes.
  • Each call to counter() increases count.

Lesson 3: Promises and Async/Await

Promises handle asynchronous operations, while async/await makes them easier.


function fetchData() {
return new Promise((resolve, reject) => {
    setTimeout(() => resolve("Data received!"), 2000);
});
}

async function getData() {
let result = await fetchData();
console.log(result);
}

getData(); // Output: "Data received!" after 2 seconds
  • fetchData() simulates a 2-second delay.
  • await pauses execution until the promise resolves.

Lesson 4: The this Keyword

The this keyword refers to the object executing the function.


let person = {
name: "Larmel",
greet: function() {
    console.log("Hello, " + this.name);
}
};

person.greet(); // Output: Hello, Larmel
  • this.name refers to the objectâ€™s property (person.name).
  • The function correctly references "Larmel".

Lesson 5: Arrow Functions

Arrow functions (=>) provide a shorter syntax and do not have their own this value.


const add = (a, b) => a + b;
console.log(add(5, 3)); // Output: 8
  • No need for the function keyword.
  • If there's only one return statement, {} can be omitted.

Lesson 6: Spread and Rest Operators

The spread operator (...) expands arrays, while the rest operator collects multiple values.


// Spread Operator
let arr1 = [1, 2, 3];
let arr2 = [...arr1, 4, 5];
console.log(arr2); // Output: [1, 2, 3, 4, 5]

// Rest Operator
function sum(...numbers) {
return numbers.reduce((total, num) => total + num, 0);
}
console.log(sum(1, 2, 3, 4)); // Output: 10
  • ... (spread) copies elements into a new array.
  • ... (rest) collects multiple arguments into an array.

Lesson 7: Destructuring Assignment

Destructuring extracts values from objects or arrays easily.


let person = { name: "Larmel", age: 19 };
let { name, age } = person;
console.log(name); // Output: Larmel

Lesson 12: Fetch API

The Fetch API allows making HTTP requests to a server.


fetch("https://jsonplaceholder.typicode.com/posts/1")
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error("Error:", error));

let x = y; // Error
