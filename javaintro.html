Lesson 1: Introduction to Java

Java is a high-level, object-oriented programming language widely used for building applications.

Hello World Program


public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Explanation:

  • public class HelloWorld: Defines a class.
  • public static void main(String[] args): The main method.
  • System.out.println("Hello, World!");: Prints text.

Lesson 2: Variables and Data Types

Java has different data types such as int, double, char, and boolean.


public class VariablesExample {
    public static void main(String[] args) {
        int age = 19;
        double price = 99.99;
        char grade = 'A';
        boolean isPassed = true;

        System.out.println("Age: " + age);
        System.out.println("Price: " + price);
        System.out.println("Grade: " + grade);
        System.out.println("Passed: " + isPassed);
    }
}

Explanation:

  • int age = 19;: Declares an integer variable.
  • double price = 99.99;: Declares a floating-point number.
  • char grade = 'A';: Declares a single character.
  • boolean isPassed = true;: Declares a boolean (true or false).

Lesson 3: Operators in Java

Operators include arithmetic, relational, and logical operations.


public class ArithmeticOperators {
    public static void main(String[] args) {
        int a = 10, b = 5;
        System.out.println("Addition: " + (a + b)); 
        System.out.println("Subtraction: " + (a - b));
    }
}

Explanation:

  • a + b: Adds two numbers.
  • a - b: Subtracts one number from another.

Lesson 4: Conditional Statements

Conditional statements allow decision-making in Java.


public class IfElseExample {
    public static void main(String[] args) {
        int num = 10;
        if (num > 0) {
            System.out.println("Positive number");
        } else {
            System.out.println("Negative number or zero");
        }
    }
}

Explanation:

  • if (num > 0): Checks if the number is positive.
  • else: Executes if the condition is false.

Lesson 5: Loops in Java

Loops allow repeated execution of code.


public class ForLoopExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            System.out.println("Iteration: " + i);
        }
    }
}

Lesson 6: Arrays in Java

Arrays store multiple values of the same type.


public class ArrayExample {
    public static void main(String[] args) {
        int[] numbers = {10, 20, 30};
        System.out.println("First Element: " + numbers[0]);
    }
}

Lesson 7: Methods in Java

Methods are reusable blocks of code.


public class MethodExample {
    public static void main(String[] args) {
        greet("Larmel");
    }
    public static void greet(String name) {
        System.out.println("Hello, " + name + "!");
    }
}

Lesson 8: Object-Oriented Programming

OOP concepts include classes and objects.


class Car {
    String brand;
    void displayInfo() {
        System.out.println("Brand: " + brand);
    }
}

public class OOPExample {
    public static void main(String[] args) {
        Car myCar = new Car();
        myCar.brand = "Toyota";
        myCar.displayInfo();
    }
}

Lesson 9: Constructors in Java

Constructors initialize objects automatically.


class Person {
    String name;
    Person(String personName) {
        name = personName;
    }
    void display() {
        System.out.println("Name: " + name);
    }
}

public class ConstructorExample {
    public static void main(String[] args) {
        Person p1 = new Person("Larmel");
        p1.display();
    }
}

Lesson 10: Inheritance in Java

Inheritance allows a class to inherit from another class.


class Animal {
    void eat() {
        System.out.println("This animal eats food.");
    }
}

class Dog extends Animal {
    void bark() {
        System.out.println("The dog barks.");
    }
}

public class InheritanceExample {
    public static void main(String[] args) {
        Dog myDog = new Dog();
        myDog.eat();
        myDog.bark();
    }
}
